Acquisition Data Analysis

This application is going to walk you through processing the data to get the results you need for your clients. It is crucial you go step by step and read ALL the instructions.

Some things to keep in mind:

A file path on your computer is like a set of directions to find a specific file. It tells your computer where the file is located in its filing system. Think of it as a digital address that helps your computer navigate through folders and directories to locate the file you're looking for. It includes the names of folders, separated by slashes, leading to the file you want to access. This matters because without the correct path, the code will not be able to call the files it needs to perform the correct functions.

Your files MUST be CSV files, any other format will break the code.

You have to download the files and save them in the proper places in order for this application to work. When you downloaded this application, all of the scripts were downloaded in a specific file path. If for some reason that path is altered, the code will break. Keep the path the same, and follow prompts to move files to ensure you do not have to re-run the code for the same client because your files were replaced with another clients information. If your downloaded files are not in your documents folder on your computer, ensure you move the entire downloaded folder into documents.

The final thing to keep in mind is that each time these scripts run, they are going to change the name of your files to be generic and saved in the same places. So if you don't rename the files and/or move them somewhere else, they will get overridden each time you use the app. Make sure you organize each of your processed files after the final download, and make sure you know which client the data belongs to or you will get confused. There is no need to hold on to old data, one time passes and you need to refresh the results for your client, you can delete old files. If you happen to need them again, the scripts will always work and you can segment out the specific dates you need.
